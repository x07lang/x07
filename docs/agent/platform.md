# Platform (x07lp)

The x07 Platform (x07-LP) — the public runtime and control-plane surface for X07 workloads and release operations, driven by the `x07lp` CLI and the `lp.*` MCP namespace — was archived in the 2026-06 refocus (see [Roadmap](../roadmap.md)).

The public `lp.*` contract schemas are frozen in the archived `x07-platform-contracts` repo. The platform engine (`x07-platform`) and the managed hosted layer (`x07-platform-cloud`) are likewise archived. There is no supported usage path from the active toolchain.
