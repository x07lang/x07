# Repository topology

This document defines the roles of the X07 repositories.

## Canonical sources

- `x07lang/x07`: canonical toolchain, stdlib, specs, and primary documentation.
- `x07lang/x07-rfcs`: canonical RFC process and RFC documents.

## Ecosystem repos

- `x07lang/x07-registry`: registry server implementation and API contract.
- `x07lang/x07-website`: website and documentation site build/deploy.
- `x07lang/x07-infra-private`: production infrastructure and operational runbooks (private).
